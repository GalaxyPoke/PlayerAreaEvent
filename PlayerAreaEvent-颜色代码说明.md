# PlayerArea 颜色代码说明

## 🎨 使用 & 符号表示颜色

PlayerArea 支持使用 **`&`** 符号来表示颜色代码，比 `§` 更方便输入！

---

## 📝 颜色代码对照表

### 颜色代码

| 代码 | 颜色 | 示例 |
|------|------|------|
| `&0` | 黑色 | `&0黑色文字` |
| `&1` | 深蓝色 | `&1深蓝色文字` |
| `&2` | 深绿色 | `&2深绿色文字` |
| `&3` | 青色 | `&3青色文字` |
| `&4` | 深红色 | `&4深红色文字` |
| `&5` | 紫色 | `&5紫色文字` |
| `&6` | 金色 | `&6金色文字` |
| `&7` | 灰色 | `&7灰色文字` |
| `&8` | 深灰色 | `&8深灰色文字` |
| `&9` | 蓝色 | `&9蓝色文字` |
| `&a` | 绿色 | `&a绿色文字` |
| `&b` | 天蓝色 | `&b天蓝色文字` |
| `&c` | 红色 | `&c红色文字` |
| `&d` | 粉色 | `&d粉色文字` |
| `&e` | 黄色 | `&e黄色文字` |
| `&f` | 白色 | `&f白色文字` |

### 格式代码

| 代码 | 效果 | 示例 |
|------|------|------|
| `&k` | 乱码 | `&k乱码文字` |
| `&l` | 粗体 | `&l粗体文字` |
| `&m` | 删除线 | `&m删除线文字` |
| `&n` | 下划线 | `&n下划线文字` |
| `&o` | 斜体 | `&o斜体文字` |
| `&r` | 重置 | `&c红色&r普通` |

## 💡 使用示例

### 1. 简单消息
```yaml
actions:
  enter: |-
    tell "&a欢迎来到出生点！"
```
**效果**：绿色文字

---

### 2. 组合颜色
```yaml
actions:
  enter: |-
    tell "&6欢迎 &e来到 &a出生点！"
```
**效果**：金色"欢迎" + 黄色"来到" + 绿色"出生点！"

---

### 3. 颜色 + 格式
```yaml
actions:
  enter: |-
    tell "&c&l警告！&r &e这是危险区域"
```
**效果**：红色粗体"警告！" + 黄色"这是危险区域"

---

### 4. 标题消息
```yaml
actions:
  enter: |-
    title "&6&l出生点" "&e安全区域"
```
**效果**：
- 主标题：金色粗体"出生点"
- 副标题：黄色"安全区域"

---

### 5. ActionBar
```yaml
actions:
  tick: |-
    actionbar "&a出生点 &7| &e安全区域"
```
**效果**：绿色"出生点" + 灰色"|" + 黄色"安全区域"

## 📋 完整示例

### 出生点区域
```yaml
spawn:
  actions:
    enter: |-
      tell "&a&l欢迎来到出生点！"
```

## 🎨 颜色搭配建议

### 1. 对比色搭配
```yaml
"&c红色 &a绿色"  # 强烈对比
"&9蓝色 &6金色"  # 经典搭配
"&d粉色 &b天蓝"  # 柔和搭配
```

### 2. 渐变色搭配
```yaml
"&c红 &6橙 &e黄"  # 暖色渐变
"&9蓝 &b青 &a绿"  # 冷色渐变
```

### 3. 主题色搭配
```yaml
# 火焰主题
"&c&l火焰 &6&o燃烧"

# 冰霜主题
"&b&l冰霜 &f&o寒冷"

# 自然主题
"&a&l自然 &2&o森林"

# 魔法主题
"&5&l魔法 &d&o神秘"
```

## ⚠️ 注意事项

### 1. 格式代码会持续
```yaml
# ❌ 错误：粗体会影响后面的文字
tell "&l粗体 普通文字"  # 全部都是粗体

# ✅ 正确：使用 &r 重置
tell "&l粗体&r 普通文字"
```

### 2. 颜色代码会覆盖
```yaml
# 后面的颜色会覆盖前面的
tell "&c红色&a绿色"  # 只显示绿色
```

### 3. 大小写不敏感
```yaml
# 这两个效果一样
tell "&A绿色"
tell "&a绿色"
```

## 💡 总结

**使用 `&` 符号的优势：**

- ✅ **更容易输入** - 不需要特殊输入法
- ✅ **更容易阅读** - 配置文件更清晰
- ✅ **更容易复制** - 不会出现乱码
- ✅ **完全兼容** - 自动转换为 Minecraft 颜色代码

**现在你可以在所有配置中使用 `&` 来表示颜色了！** 🎨
